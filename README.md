# Skills

AI agent skills for Claude Code and other AI tools.

## Available Skills

### create-skill

Build Claude Skills from scratch. Use when you want to create, build, or write a skill for AI agents.

**Install:**
```bash
npx skills@latest add ZeiProX76/skills/create-skill
```

## Usage

Each skill is a self-contained Markdown file that enhances AI agent capabilities. Install skills using the CLI:

```bash
npx skills@latest add ZeiProX76/skills/<skill-name>
```
